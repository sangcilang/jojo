# Quick Start - Chức Năng Nhân Viên

## 🚀 Đăng Nhập Nhân Viên

**Tài khoản:**
- Username: `nhanvien`
- Password: `Staff@123`

## 📋 Chức Năng Chính

### 1. Xem Danh Sách Đơn Hàng
- Tất cả đơn hàng hiển thị theo thời gian mới nhất
- Click vào đơn để xem chi tiết

### 2. Tìm Kiếm & Lọc
- 🔍 Tìm theo mã đơn hàng
- 📊 Lọc theo 7 trạng thái

### 3. Cập Nhật Trạng Thái

**Nhận tại store:**
```
Đang chuẩn bị → Sẵn sàng nhận → Hoàn thành
```

**Giao hàng:**
```
Đang chuẩn bị → Đang giao → Đã giao → Hoàn thành
```

### 4. Hủy Đơn Hàng
- Bấm nút "❌ Hủy đơn hàng"
- Xác nhận để hủy

### 5. In Hóa Đơn
- Bấm nút "🖨️ In hóa đơn"
- Cửa sổ mới mở ra để in

## 🎯 Workflow Nhanh

1. Đăng nhập → Tự động vào trang quản lý đơn hàng
2. Tìm/lọc đơn hàng cần xử lý
3. Click vào đơn → Xem chi tiết
4. Chọn hành động: Cập nhật trạng thái / Hủy / In hóa đơn
5. Hoàn tất!

## 💡 Tips

- Đơn đã hoàn thành/hủy không thể thay đổi
- Workflow animation hiển thị tiến độ trực quan
- Có thể in hóa đơn bất cứ lúc nào
- Tìm kiếm không phân biệt hoa thường

## 🔗 URL

- Trang nhân viên: `http://localhost:3000/#/staff`
- Đăng nhập: `http://localhost:3000/#/auth`
